TUTORIAL


TRIK 1 :
  1. Wajib ada pulsa minimal Rp1.
  2. Pilih "Paket RP1 12GB 5DAY *REBORN".
  3. Klik "TEMBAK PAKET".
  4. Paket ini tidak bisa ditembak berkali-kali.

TRIK 2 :
  1. Trik ini wajib bekas paketan RP.1 yang paketannya sudah hilang.
  2. Pasti kan Simcard tidak memiliki paket internet apapun di *123*7*3#.
  3. Ciri-ciri bekasan paket RP.1 adalah QOS nya Upload : 1MB dan Download : 7MB .
  4. Untuk memastikan nya, silahkan tembak paket "Gaul Unlimited Rp.1700/Hari".
  5. Jika sudah di paketkan silahkan test download sampai 35MB, jika lewat 35MB speed masih sama atau QOS tidak anjlok, berarti sukses.
  6. Jika pengen di upgrade paketan nya silahkan unreg dulu paket "Gaul Unlimited Rp.1700/Hari" nya lewat *123*7*4*2#.
  7. Setelah di unreg paket harian nya silahkan tembak ulang "Gaul Unlimited Rp.9900/Minggu" atau "Gaul Unlimited Rp.35.900/Bulan".
  8. QOS harian setelah sukses : Upload 256KB Download 3MB.
  9. QOS mingguan dan Bulanan setelah sukses : Upload 256KB Download 7MB.
  10. *QOS = Quality Of Service bisa dilihat memakai aplikasi MMD buat di PC. Jika di android saya belum menemukan buat cek QOS.
  11. Trik ini sudah lolos FUP (Fair Usage Police), jika tidak mengikuti trik ini pasti akan terkena FUP
  12. *NB : untuk cek QOS silahkan di 3G Only. Tetapi trik ini work di semua jaringan.
